xwga :: Extend WGA

Extends a user account on the guest WiFi system.

Utilizes selenium and PhantomJS in a Windows Linux server environment to manipulate the webpages that administrate guest accounts and automatically extends them. Automatically detects existing accounts, creating new accounts if the guest is not found, reading all user information from a newline-delimited>comma-delimited file.

New Release :: Linux server conversion

Completed Linux configuration, all software compiled from source code of equivalent versions used in Windows runtime. Final build includes CentOS v7.1.1503, Python 2.7.5, Selenium v2.45.0, Nodejs v0.12.6, PhantomJS v2.0.0. Updated repository with final build script.
