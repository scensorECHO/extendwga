xwga :: Extend WGA

Extends a user account on the guest WiFi system. Utilizes selenium and PhantomJS in a Windows environment to manipulate the webpages to administrate guest accounts and automatically extend them. Automatically detects existing accounts, creating new accounts if the guest is not found, reading all user information from a newline-delimited:comma-delimited spreadsheet.

Next patch :: Linux server conversion Currently upgrading the Node.js install on the Linux server in order to correctly compile PhantomJS and configure in a Linux environment to conserve company space and energy.
